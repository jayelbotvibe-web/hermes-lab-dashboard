YOU — What you click

1
[CLICK] ▶ Start Pentest

2
[auto] LUKS keyfile → mount → Docker containers → ✓ READY
3

🔍 FORENSICS LAB
1
[CLICK] ▶ Start Forensics
Background dispatch: LUKS → mount → SIFT VM → canary → ✓ READY
2
Open Hermes Desktop → switch to forensics profile → new chat
19/19 canary checks • Tools: Volatility3 · Plaso · MFT · MemProcFS · Sleuthkit · Foremost · dc3dd · hashdeep · tshark · RegRipper · via SIFT VM
Dispatch: hermes -z "task" --profile pentest  |  hermes -z "task" --profile forensics
